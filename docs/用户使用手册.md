## 1. 系统简介

### 1.1 概述

Local AI Knowledge 是一款基于 RAG（检索增强生成）技术的智能知识库问答系统，支持：

- 📚 **知识库管理**：上传文档自动解析、切片、向量化
- 💬 **智能问答**：基于知识库的 AI 问答，支持多轮对话
- 🌐 **每日热榜**：多源热点数据采集（GitHub、微博、B站等）
- 🔐 **权限管理**：完整的用户认证和角色权限体系

### 1.2 技术架构

```
前端：Vue 3 + TypeScript + Element Plus
后端：Spring Boot + Spring AI
向量库：Elasticsearch / PostgreSQL + pgvector
缓存：Redis + Caffeine
```

### 1.3 角色说明

| 角色 | 说明 | 权限范围 |
|:--|:--|:--|
| 普通用户 | 系统默认角色 | 智能问答、上传文档、个人中心 |
| 管理员 | 系统管理员 | 全部功能，包括用户管理、智能体配置、爬虫管理 |

## 2. 快速入门

### 2.1 登录系统

1. 访问系统登录页面
2. 输入用户名和密码
3. 点击「登录」按钮进入系统

> 💡 **提示**：勾选「记住密码」可保存登录信息

### 2.2 首次使用流程

```
1. 登录系统
2. 上传文档到知识库（文档管理页面）
3. 等待文档解析完成
4. 进入智能问答页面开始提问
```

## 4. 智能问答

### 4.1 界面介绍

智能问答页面分为三个区域：

```
┌──────────────────────────────────────────────────────────────┐
│  历史会话            │           智能问答                      │
│  ┌──────────────┐  │  ┌────────────────────────────────────┐ │
│  │ + 新对话     │  │  │                                    │ │
│  ├──────────────┤  │  │      欢迎使用智能问答              │ │
│  │ 会话1        │  │  │   我可以帮您基于知识库回答问题...   │ │
│  │ 会话2        │  │  │                                    │ │
│  │ 会话3        │  │  └────────────────────────────────────┘ │
│  └──────────────┘  │  ┌────────────────────────────────────┐ │
│                    │  │ 请输入您的问题...              发送 │ │
│                    │  └────────────────────────────────────┘ │
└──────────────────────────────────────────────────────────────┘
```

### 4.2 问答模式

系统提供两种问答模式：

| 模式 | 说明 | 适用场景 |
|:--|:--|:--|
| 知识库模式 | 基于上传的文档检索回答 | 需要精准答案时 |
| LLM 直答 | AI 通用知识直接回答 | 通用问题参考 |

> ⚠️ **注意**：LLM 直答模式可能存在幻觉，答案仅供参考

### 4.3 多轮对话

- **新建对话**：点击左侧「+ 新对话」按钮
- **切换会话**：点击左侧历史会话列表中的任意会话
- **删除会话**：鼠标悬停会话，点击删除图标
- **重命名会话**：鼠标悬停会话，点击编辑图标

### 4.4 答案来源

系统回答后会显示答案来源：

| 标签 | 含义 |
|:--|:--|
| 🟢 知识库回答 | 基于您上传的文档回答 |
| 🟡 网络搜索回答 | 基于网络搜索降级回答 |
| 🔵 AI 通用知识 | 基于 LLM 自身知识回答（可能有幻觉） |

点击「查看 N 个来源」可展开查看具体引用文档。

### 4.5 移动端适配

- 点击左上角按钮展开/收起历史会话
- 点击遮罩层收起侧边栏

## 5. 文档管理

### 5.1 上传文档

1. 进入「文档管理」页面
2. 点击「上传文档」按钮
3. 选择文件或拖拽文件到上传区域
4. 选择文档范围：
   - **私有**：仅自己可见
   - **公开**：所有用户可见
5. 点击「上传」

> 📁 **支持格式**：PDF、Word、TXT 等常见文档格式
> 📏 **文件限制**：单文件不超过 50MB

### 5.2 解析状态

文档上传后会经历以下状态：

| 状态 | 说明 |
|:--|:--|
| 已上传 | 文件已上传，等待解析 |
| 解析中 | 系统正在解析文档内容 |
| 入库中 | 文档正在切片并向量化 |
| 完成 | 文档已可用于问答 |
| 失败 | 解析失败，可查看日志排查 |

### 5.3 切片进度

上传过程中可实时查看切片进度：

- 进度条显示当前入库进度
- 上传完成后自动停止轮询
- 全部完成后可在智能问答中使用

### 5.4 文档操作

| 操作 | 说明 |
|:--|:--|
| 查看日志 | 查看解析详细日志 |
| 下载文档 | 下载原始文档文件 |
| 删除文档 | 删除文档及对应向量数据 |

> ⚠️ **警告**：删除后不可恢复，向量数据也会一并清除

## 6. 每日热榜

### 6.1 功能介绍

每日热榜模块展示多源热点数据采集结果，包括：

- GitHub Trending（技术项目趋势）
- 微博热搜
- 知乎热榜
- B站热门

### 6.2 数据展示

| 字段 | 说明 |
|:--|:--|
| 排名 | 热榜序号，前三名高亮显示 |
| 来源 | 数据来源平台 |
| 标题 | 热点标题（可点击跳转原链接） |
| 描述 | 热点内容摘要 |
| 热度 | 热度评分 |
| 采集时间 | 数据采集时间 |

### 6.3 数据筛选

- 按来源筛选：点击筛选栏切换不同来源
- 分页浏览：支持 20/50/100 条/页
- 刷新数据：点击刷新按钮更新数据

### 6.4 趋势图

页面底部展示近 7 天采集趋势折线图，直观展示各来源数据变化。

## 9. 常见问题

### Q1: 文档上传后多久可用？

A：一般情况下，小文档（<10MB）解析需要 1-3 分钟，大文档可能需要 5-10 分钟。可在文档列表查看解析状态。

### Q2: 为什么答案不准确？

A：请检查以下几点：
1. 确认文档已解析完成（状态为「完成」）
2. 尝试切换到「知识库模式」
3. 换一种问法重新提问

### Q3: 如何让 AI 只回答知识库中的内容？

A：使用「知识库模式」进行问答，系统会优先检索知识库内容。当知识库无相关内容时，会自动降级到网络搜索。

### Q4: 忘记密码怎么办？

A：请联系系统管理员重置密码。

### Q5: 管理员账号密码是什么？

A：管理员账号由系统部署时创建，如有疑问请联系系统管理员。
